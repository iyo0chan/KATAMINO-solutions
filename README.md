# KATAMINO_solutions

KATAMINOはGigamic社から発売されているKATAMINOは12個のペントミノを使って長方形を作るパズルゲームです。

このプログラムは、12個のピース全てを使いパズルの解を全て出力します。

## 依存ライブラリ

* matplotlib3.4.3
* numpy 1.20.3
